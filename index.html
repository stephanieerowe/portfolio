Software Engineering

SwapNest

Full-stack Android mobile application for local, location-aware item bartering between two user roles: providers and receivers. Instead of selling items for money, SwapNest lets people trade what they have for what they need.

Providers can create listings with photos, item condition, location, and tags, while receivers browse nearby items on a map, filter by category or condition, submit trade requests, and track request status. The app also includes in-app chat, push notifications, transaction history, and post-trade rating flows to keep the exchange process organized.

  • Role-based provider and receiver flows
  • Nearby discovery with maps and geolocation
  • Trade requests, chat, and realtime updates

Web Project

YooYyaa Immigration

Yooyyaa Immigration is a responsive informational website for an immigration consultancy. It presents services, team information, FAQ content, contact forms, and an admin landing page for managing submissions.

The site was designed as a public-facing marketing and lead capture tool, helping prospective clients learn about services, contact the team, and submit inquiries. It also includes a lightweight admin interface for reviewing or recording submissions.

  • Service pages and imagery for clear marketing
  • Contact form and server endpoint for lead capture
  • JSON-driven content for easy updates
  • Admin route support for managing submissions
  • Accessible, responsive layout for all screen sizes
  • Built with semantic HTML and visible focus states

Accessibility was a core goal throughout the build: semantic HTML, keyboard-friendly interaction, descriptive labels and alt text, strong contrast, and ARIA support where needed for complex widgets such as accordions.

The project was completed in a collaborative MERN-style workflow with separate frontend and backend responsibilities, code review through pull requests, and shared linting and local tooling to keep the implementation consistent.
