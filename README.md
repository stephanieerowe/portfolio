# Portfolio Starter

A clean first-commit portfolio site with:

- A home page with a hero, featured projects, skills, process, and contact section
- An about page with background, values, toolkit, and timeline
- Shared styling and small JavaScript enhancements for navigation and scroll reveals

## Files

- `index.html` - main landing page
- `about.html` - about me page
- `styles/main.css` - shared styling
- `scripts/main.js` - navigation toggle, year, and reveal behavior
